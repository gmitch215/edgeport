    writeFile

    • writeFile(
          path: string,
          data: string | Uint8Array<ArrayBufferLike>,
          opts?: SftpWriteOptions,
      ): Promise<void>

      Writes a file. data may be bytes or a string (a string is UTF-8 encoded, so callers do not build a TextEncoder). With no offset the file is created/truncated; with offset the data is written at that byte position without truncating - the basis for resuming an interrupted upload (stat the partial file, then write the remainder at its size).

      Parameters
