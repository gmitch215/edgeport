    Interface DnsSession

    A live DNS session over a single TCP (or TLS) connection.

    Obtain one from connect. A background pump reads length-prefixed responses and correlates them to pending queries by id, so several query calls can be in flight at once (RFC 7766 pipelining). It is an AsyncDisposable, so await using closes it cleanly.
