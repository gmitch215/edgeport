    Optionaltls

    tls?: "off" | "starttls" | "implicit"

    Transport security:

    • 'off': plaintext.
    • 'implicit': TLS from the first byte.
    • 'starttls': plaintext, upgraded to TLS right after the server INFO.

    Defaults to 'starttls' when the server advertises tls_required, otherwise 'off'.
